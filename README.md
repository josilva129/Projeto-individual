# WizardHub – Universo Harry Potter

## Sobre o projeto
O WizardHub é um website temático inspirado no universo de Harry Potter, desenvolvido como projeto individual da disciplina de Projeto Individual da São Paulo Tech School.
O principal objetivo do projeto é unir um tema de interesse pessoal aos conhecimentos adquiridos ao longo do semestre.

## Objetivo
O principal objetivo do projeto é aplicar, de forma prática,
os conhecimentos adquiridos ao longo do semestre nas disciplinas do curso de
Análise e Desenvolvimento de Sistemas, unindo aprendizado técnico a um tema
de interesse pessoal.

## Funcionalidades
- Cadastro de usuários
- Login e autenticação
- Dashboard com indicadores
- Sistema de login e logout
